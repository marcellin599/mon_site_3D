# Mon Site 3D — Portfolio Interactif

Une expérience web immersive avec environnement 3D, animations et navigation visuelle avancée.

---

## Description

Ce projet est un **portfolio interactif en 3D** réalisé avec [Three.js](https://threejs.org/).  
Il met en avant mes projets sous forme de cubes cliquables dans un espace 3D, avec :

- Animation et rotation des cubes
- Sol texturé et environnement stylé
- Overlay d’informations au clic
- Contrôles de caméra fluides avec OrbitControls

## Installation & lancement

1. Cloner le projet

-git clone https://github.com/marcellin599/mon_site_3D.git
-cd mon_site_3D
-Installer les dépendances (si tu utilises Vite ou npm)
-npm install
-Lancer le serveur de développement
-npm run dev
-Ouvrir le projet dans ton navigateur à l’adresse indiquée par Vite.
